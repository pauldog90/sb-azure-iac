This started as doodles on a notepad while watching TV, but my daughter loved the characters so I began turning them into comics. She wanted to be able to view them on my phone and the TV, so I built a simple website to make them accessible everywhere.

The original hosting costs started to add up, so I moved the site to an Azure Static Web App. Since I was doing that anyway, I used it as a chance to automate the infrastructure with Terraform and GitHub Actions - partly to keep costs down, partly as a bit of hands-on practice.
